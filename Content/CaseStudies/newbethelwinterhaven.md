# Modernizing Ministry Outreach

The New Bethel Winter Haven project was designed to give a growing non-profit ministry a digital platform capable of reaching their community effectively. It replaces an outdated, hard-to-maintain system with a dynamic, interactive web experience.

## Key Features
- **Real-Time Mapping:** Integrated with Leaflet.js and the GraphHopper API to provide turn-by-turn routing directly to the church without leaving the browser.
- **Live Streaming & Archives:** Integrated streaming services and a searchable sermon archive so members can stay connected remotely.
- **Donation Processing:** Seamless Cash App integration to facilitate digital giving.

## Technical Architecture
- Built on ASP.NET Core MVC/Razor Pages.
- Utilizes **ASP.NET Core SignalR** for real-time announcements during live services.
- Fully hosted on Microsoft Azure for high availability during peak Sunday traffic spikes.
- UI built with Bootstrap 5 and custom CSS for a mobile-first, highly accessible experience.
