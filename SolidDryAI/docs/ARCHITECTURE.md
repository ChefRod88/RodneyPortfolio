# SolidDry AI Architecture

## Layers
- `SolidDry.Web`: UI and API surface
- `SolidDry.Application`: business logic and orchestration
- `SolidDry.Core`: entities, enums, domain contracts
- `SolidDry.Infrastructure`: persistence and external integrations

## Phase 1 Scope
Phase 1 establishes the solution structure, domain entities, and EF Core DbContext.
