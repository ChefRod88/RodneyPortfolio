# Layer 0: Global Identity Configuration

This workspace is configured for Rodney Chery's Portfolio AI System.
It implements the Model Workspace Protocol (MWP) / Interpretable Context Methodology (ICM) described in arXiv:2603.16021.

## Workspace Root
`/IcmWorkspace`

## Folder Structure
- `_config/`: Layer 3 stable reference materials (Resume, Voice, Design System)
- `01_skill_extraction/`: Stage 1 folder (Extract requirements from job descriptions)
  - `CONTEXT.md`: Stage Contract
  - `output/`: Folder containing Stage 1 outputs
- `02_resume_comparison/`: Stage 2 folder (Compare extracted requirements to Rodney's resume)
  - `CONTEXT.md`: Stage Contract
  - `output/`: Folder containing Stage 2 outputs
- `03_interview_preparation/`: Stage 3 folder (Generate targeted talking points and interview guide)
  - `CONTEXT.md`: Stage Contract
  - `output/`: Folder containing Stage 3 outputs
